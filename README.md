# Weather App

A modern and responsive Weather Report Web Application built using React + Vite + Tailwind CSS.

This app shows real-time weather details for any city using the OpenWeatherMap API.

# Features

 •Search weather by city name
 •Current temperature in Celsius
 •Wind speed
 •Humidity
 •Cloud cover
 •Visibility
 •Sunrise time
 •Sunset time
 •Longitude & Latitude
 •Full-screen video background
 •Fully responsive for mobile and desktop

# Tech Stack

- React.js
- Vite
- Tailwind CSS
- Axios
- OpenWeatherMap API
